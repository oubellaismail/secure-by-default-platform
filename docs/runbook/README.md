# Runbook

Operational how-tos for standing up, operating, and tearing down the platform.
Start with the top-level README quickstart, then per-component notes here.

- Cluster up/down: `make cluster-up` / `make cluster-down` (tear down when idle).
- Bootstrap: `make bootstrap`.
- Break-glass: how to temporarily set a Kyverno policy to `Audit` and why.
