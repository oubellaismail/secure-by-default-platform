# Reports

Progress reports for the project, written in Markdown so they are easy to share.

## Convention

- Copy `TEMPLATE.md` for each new report.
- Name files `NN-week-N-topic.md` (e.g. `01-week-1-threat-model.md`) so they sort in order.
- One report per reporting period. Keep them short and outcome-focused.
- Fill every section; write "None" where a section does not apply rather than deleting it.
